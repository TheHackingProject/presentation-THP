The Hacking Project

Éducation digitale et communautaire

Le problème, et notre objectif

  • Le problème : Félix a toujours voulu apprendre à coder. Mais il était confronté à soit des coding bootcamp onéreux, soit à des sites où les étudiants étaient livrés à eux-mêmes..
  • Notre but : nous voulons proposer une formation de code gratuite et communautaire.

Le pitch, en quelques lignes

  • The Hacking Project est un coding bootcamp de trois mois entièrement gratuit et ouvert à tous. Il a 2 formes :
    • Onsite : 6 personnes se retrouvent tous les jours pour faire le cursus
    • Une version online est prévue, et les étudiants pourront suivre la formation à distance

Les inspirations

Ou Comment accéder à une formation code gratuite et de qualité ?

Qui fait déjà cela ?

  • Les coding bootcamp traditionnels (ex: Le Wagon, Hack Reactor)
    • Bon suivi des élèves, mais onéreux
  • Les bootcamps en ligne gratuits (ex: The Odin Project, FreeCodeCamp)
    • Gratuits, mais élèves livrés à eux-même

Comment THP innove?

Deux axes : réinvention de l'apprentissage, et communauté

Réinvention de l'apprentissage

  • L'éducation est gratuite et privilégie l'auto-formation : les ressources en ligne pour le code sont nombreuses, il faut juste faire le travail de curation que nous exécutons.
  • L'apprentissage se fait via projets qui mettent en pratique les connaissance théoriques acquises.

Communauté

  • Aider les autres permet de s'améliorer, ainsi les élèves n'hésitent à se donner un coup de main entre eux.
  • Un système de mentoring est mis en place où des professionnels passent un peu de temps avec les élèves, via un système donnant-donnant.

Nos méthodologies

Comment la formation se déroulera ?

Les cours

  • La formation donne les compétences pour être développeur full-stack web. HTML / CSS / Ruby / Rails / JavaScript / Jquery / SQL
  • La partie théorique prend un florilège des ressources en ligne disponibles gratuitement
  • À ces théories sont ajoutés des projets qui mettent en pratique les cours

Exemples de projets

  • Un projet de scrapping qui permet de prendre des données d'un site
  • Un projet "Twitter", où les élèves doivent créer un Twitter Spambot et lui faire avoir un maximum de followers
  • Un jeu d'échecs en ligne de commande
  • En fin de formation, les élèves seront capables de recréer Airbnb
